# Meus Políticos — Kit SVG v3 (visual fiel ao PNG aprovado)

Este pacote corrige a tentativa anterior.

Importante: para manter o logo exatamente igual ao PNG aprovado, os SVGs principais incorporam o PNG recortado como imagem raster dentro do SVG. Isso preserva proporção, textura, cortes, peso visual e desenho original.

Arquivos principais:
- 01-logo-principal-visual-fiel.svg
- 02-logo-principal-fundo-branco-visual-fiel.svg
- 03-logo-monocromatico-preto-visual-fiel.svg
- 04-logo-negativo-branco-fundo-verde-petroleo-visual-fiel.svg
- 05-icone-principal-visual-fiel.svg
- 06-icone-monocromatico-preto-visual-fiel.svg
- 07-icone-negativo-branco-fundo-verde-petroleo-visual-fiel.svg
- 08-prancha-aplicacoes-visual-fiel.svg

Atenção: isso é SVG visualmente fiel, não vetor geométrico 100% editável por path. Para chegar a vetor puro idêntico, é necessário redesenhar manualmente no Illustrator/Figma por cima do PNG-mestre.
